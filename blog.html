How we can find the Current year Revenue till last month?

In order to calculate Revenue/Expense/Sales Values of Current year till last month. Please refer to the formula which is explained below. The below method is used to find the expected Values.

Ex: If we are calculating for next year’s first month, then it will show last Year's total sales.

To proceed further we need to know about Dax.

Let’s see what is DAX.

Data Analysis Expression (DAX) is a formula expression language used in different BI & Visualization tools. DAX is also called function language, where the full code is kept inside a function and it’s helpful for using Aggregate, Logical, Date (SUM, COUNT, MIN, MAX, AVG, If, Or, Date, Year, Month,…). Power BI DAX could be a combination of functions, operators & constants which will be utilized in a formula, or expression, in order to calculate and come one or additional values. DAX helps you in producing new info from the existing information of your model.

DAX Formula:

YEAR ():

Returns the year of a date as a four-digit integer in the range 1900-9999. Ex: 2022

MONTH(<datetime>)

                Returns the month as a number from 1 (January) to 12 (December).

TODAY ():

       Returns the current date. Ex: 02/24/2022


EOMONTH(<start_date>, <months>)

Returns the date in date-time format of the last day of the month, before or after a specified number of months. Use EOMONTH to calculate maturity dates or due dates that fall on the last day of the month. Ex: 01/31/2022

 

VAR <name> = <expression>  

       Stores the result of an expression as a named variable, which can then be passed as an argument to other measure expressions. Once resultant values have been calculated for a variable expression, those values do not change, even if the variable is referenced in another expression.

Ex: Var a = sum(sales) 

       Var b = calculate (a, year (today ( )))

Return b

CALCULATE (<expression>,<filter1>,<filter2>…)  

                Evaluates an expression in a context that is modified by the specified filters.

Ex: Calculate (Sum (Sales), All(Date))
 

SUM(<column>)  

Adds all the numbers in a column. Ex: Sum (Sales)

 

PARALLELPERIOD (<dates>,<number_of_intervals>,<interval>)  

Returns a table that contains a column of dates that represents a period parallel to the dates in the specified dates column, in the current context, with the dates shifted a number of intervals either forward in time or back in time.

Ex: Calculate (Sum (Sales), PARALLELPERIOD(Tablename[AccountingDate]),1,Year,Month,Quarter) 

Now, we can resolve this, using the above DAX function.

 

In order to calculate Revenue/Expense/Sales Values of Current year till last month. Please refer to the formula which is explained below. The below method is used to find the expected Values.

Ex: If we are calculating for next year’s first month, then it will show last Year total sales.

This_Year_Till_Lastmonth =

var a=year(EOMONTH((TODAY()),-1))

var b=month(EOMONTH(TODAY(),-1))

Return

CALCULATE (SUM (Tablename[AmountRevenue]),YEAR(Tablename[AccountingDate])=a,

MONTH (Tablename[AccountingDate])<=b)

 

In this formula:  var a = year(EOMONTH((TODAY()),-1)) will return 2022

                                var b=month(EOMONTH(TODAY(),-1)) will return 1

Return

CALCULATE (SUM (Tablename[AmountRevenue]),YEAR(Tablename[AccountingDate])=a,
